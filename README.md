# 🎵 イントロクイズ (Intro Quiz Web)

YouTube MusicやSpotifyのプレイリストからランダムに楽曲を取得し、イントロ（最初の数秒）だけを聴いて曲名を当てる、ブラウザ完結型のWebアプリケーションです。現在、**YouTube Musicモード**をメインとして稼働しています。

---

## 🌟 主な機能

* **自分のプレイリストで遊べる**: YouTube Musicで自分が作成・保存したプレイリストから出題。
* **カスタムURL機能**: YouTube上に公開されている任意のプレイリストURLを入力して遊ぶことができます。
* **便利なショートカット**: 「アイドルマスター全曲」などの巨大プレイリストをワンクリックでセット可能。
* **多彩なゲームモード**:
    * **10問モード / 25問モード**: スコアアタック形式。
    * **エンドレスモード**: ライフ3つでどこまでいけるか挑戦。
* **イントロの長さ調整**: 3秒、5秒、10秒、15秒、30秒から選択可能。
* **コンボシステム**: 連続正解で獲得スコアの倍率がアップ（最大3倍）。

---

## 🎮 遊び方

1.  GitHub Pages等で公開されているURLにアクセスします。
2.  「YouTube Music」を選択し、Googleアカウントでログインします。
    * ※初回のみ、Google Cloud Consoleでの「OAuth クライアント ID」の発行・設定が必要です。
3.  遊びたいモードとイントロの秒数を選択します。
4.  「URLで遊ぶ」にYouTubeプレイリストのURLを貼り付けて「開始」するか、ショートカットボタンを押します。または「自分のプレイリストを選ぶ」から、自身のアカウントのリストを選択します。
5.  音が鳴り始めたら、4つの選択肢から正解を選びます（キーボードの1〜4、または矢印キー+Enterでも操作可能）。

### ⚠️ 重要：広告とYouTube Premiumについて
本アプリはYouTube公式プレイヤーを制御しているため、無料アカウントの場合はイントロの代わりに広告が流れる場合があります。

* **YouTube Premium加入者の場合**:
    ブラウザの「サードパーティCookie」を許可しないと、Premium情報が読み取れず広告が流れます。
    * **PC (Chrome等)**: アドレスバー横のサイト設定から「サードパーティ Cookie」を **「許可」** してください。
    * **スマホ (Safari)**: 設定アプリ > Safari > **「サイト越えトラッキングを防ぐ」をオフ** にしてください。
* **非加入者の場合**:
    「Brave」ブラウザや「uBlock Origin」等の拡張機能を使用することで、より快適にプレイ可能です。

## 🛠 実装ロジックと技術スタック

本アプリは、ビルドツールを必要としない **単一のHTML/CSS/JavaScript (Vanilla JS)** で実装されています。

### 1. 認証システム (OAuth 2.0 with PKCE)
外部サーバーを持たない静的ページで安全にログインを実装するため、**PKCE (Proof Key for Code Exchange)** を用いたフローを自前で実装しています。
* ランダムな文字列（Code Verifier）を生成し、ハッシュ化して認証画面に送信。
* リダイレクト後のAuth CodeとVerifierを使ってアクセストークンを直接取得。
* トークン期限切れ時はリフレッシュトークンを用いてバックグラウンドで自動更新します。

### 2. 楽曲の取得と出題ロジック
* **取得**: YouTube Data API v3 (`/playlistItems`) を使用し、楽曲リストを配列に格納。
* **出題**: 
    1. 未出題の曲から「正解」を1つ選出。
    2. 残りの楽曲リストをシャッフルし、「ダミー」を3つ抽出。
    3. 正解＋ダミーを混ぜて4択ボタンを生成。
* **スコア**: 回答速度とコンボ数（最大×3.0）に応じて算出。

### 3. YouTubeプレイヤーの制御
画面外に透明なiframe（`YT.Player`）を配置して音声を再生。
* **再生遅延対策**: プレイヤーのステータスが `1 (Playing)` に変わったイベントを検知してからカウントダウンを開始するよう制御し、通信ラグによる時間ロスを防いでいます。
* **エラーハンドリング**: 地域制限等で再生不可の動画に当たった場合は自動検知してスキップします。

## 📜 開発の軌跡：Spotify版の凍結について

本アプリは当初、Spotify APIを利用したイントロクイズとして開発がスタートしました。

**【Spotify版の挫折】**
初期は `Spotify Web Playback SDK` 等を利用してユーザーのデバイスを遠隔操作する仕組みでしたが、Spotify側のAPI制限（開発モードでの403 Forbidden多発やマーケット制限など）により、安定動作が困難となりました。

**【YouTube Musicへの移行】**
上記問題を解決するため、公式のiframe APIが安定しており、膨大な公開プレイリストにアクセスできる **YouTube Data API v3** へメインエンジンを移行しました。
※コード内には過去のSpotify用ロジック（`fetchSpTr`等）が残されていますが、現在はUI上でグレーアウトし「開発凍結」としています。
